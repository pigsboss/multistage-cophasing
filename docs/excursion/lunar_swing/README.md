# 🌙 LUNAR-SWING: 地月共振摆动轨道与大气穿越维持技术研究

## 🎯 项目概述

**LUNAR-SWING** 是 MCPC 框架下的首个专项研究项目，旨在探索地月三体系统中的共振摆动轨道。该轨道具有以下特征：

- **近地点**：低于 200 km（LEO 轨道高度），穿越地球大气层上部
- **远地点**：接近月球轨道，利用月球引力进行轨道能量调节
- **轨道维持**：通过月球引力补偿大气阻力损耗，实现长期轨道稳定性
- **共振特性**：与月球轨道形成稳定共振比（如 2:1、3:2 等）

## 🔬 科学目标

1. **轨道设计验证**：证明地月共振摆动轨道的存在性与稳定性
2. **维持机制验证**：验证"大气穿越-月球引力补偿"轨道维持新概念
3. **软件能力拓展**：测试 MCPC 框架处理复杂三体摄动问题的能力边界
4. **算法基础建立**：为后续 L3 级日地三体轨道搜索奠定数值算法基础

## 🏗️ 技术架构

本项目在 MCPC 现有架构基础上进行以下拓展：

### 1. 通用三体引力模型
- 重构硬编码的 CRTBP 实现为通用基类
- 支持任意双主天体系统配置
- 提供地月、日地等预设系统

### 2. 增强摄动模型库
- 地球非球形引力场（J2-J6）
- 地球大气阻力模型（指数模型）
- 月球外大气层模型
- 太阳光压（为日地系统准备）

### 3. 轨道设计算法
- 共振轨道搜索算法（打靶法）
- 摆动轨道优化器
- 稳定性分析工具

### 4. 专用仿真场景
- 地月摆动轨道长期演化仿真
- 大气穿越过程高精度模拟
- 燃料消耗与轨道维持策略评估

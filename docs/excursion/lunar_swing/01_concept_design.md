# 01. 概念设计：摆动轨道物理原理

## 1.1 核心概念

**地月共振摆动轨道** 是一种特殊的周期轨道，其核心特征如下：

### 轨道几何特征
- **低近地点**：200 km 以下，穿越地球大气层热层上部
- **高远地点**：接近月球轨道（~380,000 km）
- **大偏心率**：e > 0.9，轨道高度椭圆
- **三维摆动**：轨道平面与地月平面有倾角，形成三维摆动运动

### 动力学特征
- **共振锁定**：轨道周期与月球周期成简单整数比（如 2:1、3:2）
- **能量交换**：近地点大气阻力损失的能量由月球引力在远地点补偿
- **长期稳定**：在理想模型中可实现无燃料轨道维持

## 1.2 物理原理

### 1.2.1 三体引力场中的运动
在 CRTBP 框架下，航天器受到三个引力源的作用：
1. **地球引力**：主导近地点附近的运动
2. **月球引力**：主导远地点附近的运动
3. **太阳引力**：长期摄动源（在初步研究中忽略）

运动方程在旋转坐标系中表示为：
```
ẍ - 2ωẏ - ω²x = -∂U/∂x
ÿ + 2ωẋ - ω²y = -∂U/∂y
z̈ = -∂U/∂z
```
其中 U 为有效势能，包含引力势和离心势。

### 1.2.2 大气穿越机制
近地点大气穿越产生阻力：
```
F_drag = -½ ρ(h) C_d A v_rel² (v_rel/|v_rel|)
```
其中：
- ρ(h)：大气密度，随高度指数衰减
- C_d：阻力系数（~2.2）
- A：航天器截面积
- v_rel：相对于大气的速度

### 1.2.3 月球引力补偿
在远地点附近，月球引力对轨道能量的影响：
```
ΔE_moon = ∫ (F_moon · v) dt
```
通过精心设计轨道相位，可使 ΔE_moon ≈ ΔE_drag，实现能量平衡。

## 1.3 轨道族分类

### 1.3.1 按共振比分
- **2:1 共振**：航天器每绕地球2圈，月球绕1圈
- **3:2 共振**：3:2 比例，轨道周期约18.2天
- **1:1 共振**：地球同步类型，但远地点达月球距离

### 1.3.2 按倾角分
- **平面轨道**：在地月平面内运动
- **三维轨道**：倾角 5°-30°，形成三维摆动
- **极轨道**：倾角 > 60°，覆盖全球观测

## 1.4 技术挑战

### 1.4.1 动力学建模挑战
1. **多尺度问题**：近地点（分钟级）与远地点（天级）时间尺度差异大
2. **强非线性**：大偏心率轨道对初始条件敏感
3. **摄动耦合**：地球非球形、大气、月球引力等摄动相互耦合

### 1.4.2 数值计算挑战
1. **积分稳定性**：长期积分（数年）的数值误差累积
2. **打靶法收敛**：高维参数空间的局部极小值问题
3. **计算效率**：高精度轨道设计需要大量计算

### 1.4.3 工程实现挑战
1. **热控问题**：大气穿越产生气动加热
2. **通信中断**：近地点可能的地面站遮挡
3. **轨道维持**：实际摄动与模型差异的补偿

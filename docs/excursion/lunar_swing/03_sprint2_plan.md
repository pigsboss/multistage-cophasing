## 技术要点

### STM 计算实现
使用变分方程：
```
dΦ/dt = A(t) · Φ(t),  Φ(0) = I
```
其中 A(t) 是动力学雅可比矩阵。

### 打靶法收敛判据
||x(T) - x(0)|| < tol (默认 1e-8)

### 初始猜测策略
基于二体共振轨道根数：
- a = (μ_earth · (T_res/2π)^2)^(1/3)
- e = 1 - r_p/a
